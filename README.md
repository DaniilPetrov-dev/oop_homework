# OOP Calculator

## Учебный проект для закрепления и демонстрации владения  
## навыками объектно-ориентированного программирования.

В коде реализованы два калькулятора: для подсчёта денег и калорий.

Калькуляторы умеют:
- Сохранять новую запись о расходах (приёме пищи) методом add_record()
- Считать, сколько денег потрачено (калорий съедено) сегодня методом get_today_stats()
- Определять, сколько ещё денег можно потратить сегодня в рублях, долларах или евро — метод get_today_cash_remained(currency)
- Определять, сколько ещё калорий можно/нужно получить сегодня — метод get_calories_remained()
- Считать, сколько денег (калорий) потрачено за последние 7 дней — метод get_week_stats() 

У калькуляторов много пересекающихся функций, они заложены в родительский класс Calculator.  
Чтобы было удобнее создавать записи, для них создан отдельный класс Record.
